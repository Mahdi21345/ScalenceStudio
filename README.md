- **Frontend**: React 18 + TypeScript + Vite
- **Styling**: Tailwind CSS + shadcn/ui
- **Animations**: Framer Motion (optimisées mobile)
- **Backend**: Express.js + TypeScript
- **Déploiement**: Netlify + Replit
